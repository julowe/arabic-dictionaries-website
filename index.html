Arabic Dictionaries

This repository tries to gather some rare and yet useful dictionaries that have fallen in the public domain

TL;DR: I just want Arabic dictionaries on my phone

Get an app that can load StarDict or mdict formatted files. I've tried these apps on iPhone, they may have Android equivalents, I haven't looked:

  1. GoldenDict-ng (for MacOS, Windows, and Linux) to be the best desktop dictionary application. It is the "next generation" of GoldenDict, which is no longer maintained.
  2. Dicty (for iPhone) is a free dictionary app for iPhone that can load StarDict formatted files. Not sure about other formats.
  3. Dictionary Universal (for iPhone) is $6, but I got it because Dicty wasn't returning all the Sanskrit words when I typed them using English letters, and this app did. Not sure if it is worth it for Arabic dictionary files, try the free app first. This app does work well (in my limited testing) with Arabic dictionary files.
Then download the StarDict dictionary files from the downloads page, and load them into the app of your choice.

What's this site about?

This site has useful Arabic dictionaries that have fallen in the public domain.

Lane's Lexicon has been made available as a machine readable format by the Perseus project and further enhanced by the Alpheios project.

Note from previous developer: There exist (to my knowledge) two android versions on the PlayStore:
  1. Lane's Arabic Lexicon
  2. Muggy's DictBrowser.

While the former is a paid app that does not even have an ad-version, the latter is available as a paid and an ad-supported app. Unfortunately there is no computer version for look-ups with free open-source software (except for the web versions -- perseus, alpheios)

Dictionary Formats

On this site, you will be able to download versions of these dictionaries in these open-formats:

  1. Babylon (*.bgl)
  2. Comma-Separated-Value (*.csv)
  3. DictionaryForMIDs (DFM) (*.jar)
  4. Dictd (*.dict.dz)
  5. MDict (*.mdx | *.mdd)
  6. Aard2 Slob (*.slob)
  7. StarDict (*.ifo | *.idx | *.dict.dz)
  8. XDXF (*.xdxf)

Note that most (?) formats are created from the GoldenDict PC version. As for the Android version I am not sure.
